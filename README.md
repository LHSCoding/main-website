# Largo High School Coding Club's website!
### Link to website: https://largo.hackclub.com
## Language and Framework
* Framework: Hugo
* Theme: Docsy
## Development
```bash
hugo server
```
* You may need to update Hugo if preinstalled (like Fedora)!
### Debugging (if it won't start at first)
```bash
hugo mod graph
```
